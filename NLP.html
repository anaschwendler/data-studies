import nltk

1. Tokenização¶

1.1 Tokenização em setenças¶

Token um pedao de um todo, ento:

  • uma palavra um token em uma sentena;
  • uma sentena um token em um paragrafo.

Logo separar um paragrafo em sentenas tokenizar sentenas.

In [2]:
from nltk.tokenize import sent_tokenize

f = open('sample.txt')
dataset = f.read()

sentence_tokenized = sent_tokenize(dataset)
print(sentence_tokenized)

1.5 Filtrando stopwords¶

Stopwords so palavras que geralmente no contribuem para o significado de uma sentena.

In [9]:
from nltk.corpus import stopwords

portuguese_stops = set(stopwords.words('portuguese'))
words = first_sentence_word_tokenized_without_punctuation

words_without_stop = [word for word in words if word not in portuguese_stops]
